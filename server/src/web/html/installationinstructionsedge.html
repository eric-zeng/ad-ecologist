Step 1: Requirements

Please check these requirements before starting. These may not apply to you, but if they do, please make sure you follow the instructions.

  • Is this the main browser you use for browsing the web? If not, please use switch to that browser to take the study.
  • Disable any ad blockers on this browser if you currently have any turned on so that we can collect data on the ads you see. See section "Disabling Ad Blockers" below if you need more instructions.
  • Please do not use incognito or private browsing mode during the the study.
Step 2: Enable extensions from other stores

You must allow Microsoft Edge to install extensions from the Chrome Web Store, where our extension is hosted.

First, open the Edge extensions page by clicking the three dots in the top right of the browser, and click "Extensions".

Then, click "Manage Extensions".

Lastly, click "Allow extensions from other stores." in the left sidebar, if it is not already on.

Step 3: Install extension

Click on the button below to install our browser extension. After you have installed the extension, a new tab with instructions will automatically open. Please follow the instructions there to continue.


Once you finish the study, we will give you instructions on how to remove the extension.

Disabling Ad Blockers

Ad blockers will prevent the study from working. If you have any of these ad blockers installed, please disable them while taking this study. You may re-enable them after completing the study.

  • Adblock Plus
  • AdBlock
  • uBlock Origin
  • Privacy Badger
  • AdGuard
  • Any other ad blockers

To disable all ad blockers and privacy add-ons, first navigate to "edge://extensions":

If the ad blocker is currently on, toggle it to the "off" state:

Once all your ad blockers and privacy add-ons are turned off, you will no longer see their logos in the top right of your browser. You may now proceed with the study.
